## هدف
هدر و فوتر سایت ترخیصان به سبک «۳ بعدی شناور / گلس‌مورف» سایت neginkazemian بازطراحی شوند، اما با حفظ تمام محتوای فعلی (لینک‌ها، لوگو، مقالات، CTA ترخیصان‌یار، اطلاعات تماس) و با پالت رنگی فعلی پروژه:
- `--primary: 225 89% 19%` (سرمه‌ای عمیق)
- `--accent: 160 84% 32%` (سبز)
- پس‌زمینهٔ روشن: درجات نرم primary-lighter + accent-lighter

سبک فعلی پروژه (سرمه‌ای سنگین در فوتر، هدر ساده با border) کنار گذاشته می‌شود.

## نمای کلی طراحی (الهام‌گرفته از مرجع)

### هدر — پیل شناور
```text
┌────────────────────────────────────────────────────────────────────┐
│  page top, با gradient blob ملایم در پس‌زمینه                       │
│                                                                    │
│   ╭──────────────────────────────────────────────────────────╮     │
│   │ [CTA pill]   نرخ ارز · بلاگ · خدمات · خانه ...  │ TARKHISUN│   │
│   ╰──────────────────────────────────────────────────────────╯     │
│            ↑ pill شناور با backdrop-blur و سایه نرم                │
└────────────────────────────────────────────────────────────────────┘
```
- `position: sticky; top: 16px`، عرض `max-w-6xl mx-auto`، `rounded-full`، `bg-background/70 backdrop-blur-xl`، `border border-white/40`، `shadow-[0_10px_40px_-10px_hsl(var(--primary)/0.25)]`.
- آیتم فعالِ منو داخل یک pill با گرادینت `from-primary to-accent` و متن سفید.
- لوگو سمت راست + بَج گرادینت کوچک (دایره با آیکن `Sparkles` در گرادینت primary→accent).
- دکمهٔ CTA سمت چپ: «تماس با ما» به‌صورت pill گرادینت accent با سایهٔ رنگی.
- موبایل: همان پیل اما فشرده؛ دکمهٔ همبرگری شیشه‌ای؛ منو به‌صورت کارت drop-down شناور با همان استایل.

### فوتر — کارت بزرگ گلس‌مورف

```text
┌─────────────────────────────────────────────────────────────────┐

│  │  [↑] [☎] [✈] [IG]                       ترخیصان           │  │

```
- پس‌زمینهٔ بخش فوتر: گرادینت نرم `from-primary-lighter via-background to-accent-lighter` به‌جای پر‌رنگِ فعلی.
- خود فوتر یک کارت گرد بزرگ (`rounded-3xl`، `bg-background/80 backdrop-blur-xl`، سایهٔ نرم رنگی، حاشیهٔ نازک `border-white/50`).
- ردیف بالا: چپ آیکن‌های گرد گرادینت (back-to-top، تلفن، تلگرام/ایمیل، اینستاگرام)، راست برند با بَج گرادینت کوچک.
- هر ستون: یک کارت داخلی با عنوان و بَج گرادینت گوشهٔ بالا، و هر آیتم به‌صورت یک «pill row» شیشه‌ای (`rounded-2xl bg-white/60 backdrop-blur px-4 py-2.5 hover:bg-white/90`).
- CTA ترخیصان‌یار حفظ می‌شود ولی استایلش به یک pill گرادینت primary→accent با سایهٔ glow تغییر می‌کند.
- نوار پایین: copyright + OctanDev + نمادک «اعتماد الکترونیکی» (badge کوچک سمت چپ، شبیه مرجع).

## تطابق رنگ
- جایی که مرجع از peach/pink استفاده کرده → از `primary` (سرمه‌ای) به‌عنوان رنگ اصلی و `accent` (سبز) برای CTA و آیتم فعال استفاده می‌شود.
- پیل آیتم فعال: `bg-gradient-to-l from-primary to-accent text-primary-foreground`.
- بَج‌های گرد گرادینت: همان گرادینت با سایهٔ رنگی `shadow-[0_8px_20px_-4px_hsl(var(--accent)/0.4)]`.
- زمینهٔ روشن: `bg-gradient-to-b from-primary-lighter/40 via-background to-accent-lighter/40`.

## محتوای حفظ‌شده (بدون تغییر)
- لینک‌های هدر: خانه، خدمات، نرخ ارز، جستجوی تعرفه، بلاگ، ترخیصان‌یار، تماس.
- ستون‌های فوتر:
  - **اطلاعات تماس**: شماره، ایمیل، آدرس بندرعباس.
  - **لینک‌های سریع**: همان ۶ آیتم فعلی.
  - **مقالات پربازدید**: همان ۴ آیتم فعلی.
- CTA «ترخیصان‌یار» با لینک تلگرام.
- متن کپی‌رایت و امضای OctanDev.
- لوگوی محلی `tarkhisun-logo.png` در هدر (40px) و فوتر (48px) — مطابق memory.

## فایل‌هایی که تغییر می‌کنند
- `src/components/Header.tsx` — بازنویسی markup با همان لینک‌ها در ساختار پیل شناور + منوی موبایل کارت‌شناور.
- `src/components/Footer.tsx` — بازنویسی markup در قالب کارت گلس‌مورف ۳بعدی با همان داده‌ها.
- `src/index.css` — افزودن کلاس‌های کمکی کوچک:
  - `.glass-card` (پس‌زمینهٔ شیشه‌ای + blur + border + shadow)
  - `.pill-nav-active`, `.pill-row`, `.icon-badge-gradient`
- بدون تغییر در `tailwind.config.ts` (همان توکن‌ها کافی است).

## نکات فنی و سازگاری
- پنل‌های مدیریت (`TSDashboard`, `TSCards`, `TSCardUser`) از Header/Footer سایت اصلی استفاده نمی‌کنند، پس بدون اثر باقی می‌مانند.
- کلاس‌های `text-persian` و سلسله‌مراتب `<strong>` برای SEO حفظ می‌شوند (مطابق memory).
- contrast: متن روی پس‌زمینهٔ روشن از `text-primary` و `text-foreground` استفاده می‌شود تا حداقل ۴.۵:۱ تأمین شود.
- چون هدر sticky و شناور است، `index.html` SSR snapshot تحت تأثیر قرار نمی‌گیرد (فقط مارک‌آپ React).
- بدون افزودن کتابخانه‌ای جدید (بدون three.js / framer-motion). افکت «۳ بعدی» با backdrop-blur + shadow رنگی + border نیم‌شفاف ساخته می‌شود.
